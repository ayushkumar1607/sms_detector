# 📩 Spam Detector using Machine Learning (NLP + Naive Bayes)

This project is a **Machine Learning based Spam Detection System** that uses **Natural Language Processing (NLP)** techniques and the **Naive Bayes algorithm** to classify messages as **Spam** or **Not Spam (Ham)**.  
It supports classification of **SMS, text messages, and email content**.
The application is **successfully deployed on Streamlit Cloud**
